## 📅 2026-05-18 (Day 4) — 백업 시스템 + 인증 개선

### 백업 시스템 강화
- 백업 최대 개수: 10개 → **30개**
- **일별 자동 백업**: KST 기준 하루 첫 실행 시 자동 백업
- 자동 백업 파일명에 `(auto backup)` 표시
- 오프라인 시 플래그 저장 → 온라인 복귀 시 동기화 후 백업
- Dropbox에서 같은 날 백업 존재 확인 → 중복 방지

### 인증 시스템 전면 개편
- Dropbox OAuth: **Implicit Flow → PKCE Authorization Code Flow**
- Refresh token 발급 → access token 만료 시 **자동 갱신**
- 모든 API 호출에서 401 발생 시 자동 토큰 갱신 후 재시도
- 앱이 갑자기 튕기면서 재로그인을 요구하는 문제 해결

### 기타
- 제목 입력란 브라우저 자동완성(autocomplete) 비활성화
